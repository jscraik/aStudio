## üéØ Component Variants

### Size Variants

Most components support consistent size variants:

- `sm` - Small, compact size
- `md` - Medium, default size
- `lg` - Large, prominent size

### Visual Variants

Common visual variants across components:

- `ghost` - Transparent background, visible on hover
- `outline` - Border with transparent background
- `solid` - Filled background

### State Variants

Interactive components support these states:

- `default` - Normal, interactive state
- `active` - Currently selected or pressed
- `disabled` - Non-interactive state
- `loading` - Processing state

## ‚ôø Accessibility Guidelines

### Color Contrast

All color combinations meet WCAG AA standards:

- Text on backgrounds: minimum 4.5:1 ratio
- Large text: minimum 3:1 ratio
- Interactive elements: minimum 3:1 ratio

### Focus Management

- All interactive elements are keyboard accessible
- Focus indicators are clearly visible
- Focus order follows logical tab sequence

### Screen Reader Support

- Semantic HTML elements used where appropriate
- ARIA attributes provided for complex interactions
- Alternative text for icons and images

## üé® Usage Examples

### CSS Custom Properties

```css
.my-component {
  background-color: var(--foundation-bg-dark-1);
  color: var(--foundation-text-dark-primary);
  border: 1px solid var(--foundation-accent-blue);
  padding: var(--spacing-md);
  border-radius: var(--radius-lg);
}
```

### Tailwind Classes

```tsx
<div className="bg-[var(--foundation-bg-dark-1)] text-[var(--foundation-text-dark-primary)] p-4 rounded-lg">
  Content with design tokens
</div>
```

### Theme Utilities

```tsx
import { colors, spacing } from "@chatui/ui";

const styles = {
  backgroundColor: colors.bg.dark1,
  padding: spacing.md,
};
```

## üîß Customization

### Override CSS Variables

```css
:root {
  --foundation-accent-blue: #0066cc; /* Custom blue */
  --foundation-bg-dark-1: #0a0a0a; /* Darker background */
}
```

### Component-Specific Overrides

```tsx
<Button className="bg-purple-500 hover:bg-purple-600" style={{ "--button-radius": "12px" }}>
  Custom Button
</Button>
```

The design system ensures consistency while providing flexibility for customization and theming.
