# Storybook Taxonomy (Design System)

**Owner:** Jamie Scott Craik (@jscraik)  
**Last updated:** 2026-01-24  
**Scope:** Storybook navigation structure for the design system.

## Navigation Structure

- **Foundations**
  - Colors
  - Typography
  - Spacing
  - Radius
  - Shadow
- **Components**
  - Button
  - Input
  - Select
  - Dialog
  - Tooltip
  - Tabs
- **Patterns**
  - Form layouts
  - Empty states
  - Authentication screens
  - Onboarding flows

## Rules

- Stories should be tagged to match this structure.
- Foundations stories should show tokens and usage guidance.
- Component stories should include required states (loading, empty, error, etc.).
